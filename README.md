# Blue Archive Auto Script

## Features

- [x] **小组** 自动打开小组，并签到
- [x] **邮箱** 一键领取邮箱全部内容
- [x] **工作任务** 一键领取邮箱全部内容
- [x] **日程** 根据所配置的学校课程，一键完成全部内容
- [x] **咖啡厅** 邀请学生,学生互动好感,领取奖励
- [x] **商店** 支持基物品商店16个商品自动购买
